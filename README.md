# KQL-Detections

As a SOC analyst, I'm constantly asking questions like, How could we have detected this earlier? or how can i really validate if this a real attack. This repository is where I explore the answer through KQL detections for Microsoft Sentinel and Defender XDR (Since im very much working closely with MDE and Microsoft Sentinal nowadays). Every query is something I've written to better understand attacker techniques, improve detection coverage, and build the mindset of a detection engineer rather than just an alert investigator.

The codes/informations i have saved in my repository is what i have created by far with the help of claude AI and Gemini. And im currently in the midst of repairing/improvising the detection rules by testing it right on the tool (with real time alert detections), so its far more efficient than how it is at the moment.
